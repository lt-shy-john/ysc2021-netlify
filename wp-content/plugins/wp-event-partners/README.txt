== Description ==

WordPress plugin for Conference Management - WPEventPartners. Event, Meetup, Webinar, Virtual or Offline Conferences  plugin to help create landing pages with schedule, speakers, sponsors, of your conference and events.

= Features of WPEventPartners = 
We help you build an Event and Conference website. Our plugin will cope with all your different needy elements required on your WordPress website. The powerful customization we offer will help you build an interactive feature-rich website. The plugin is built such that it would help you manage sessions, speakers, schedules for your conference.

* Best fit to create landing page for your conferences and events.
* Show event session with all details of conferences.
* Mobile first website for your conference and event.
* Timezone option for virtual event organizers.


= Available Shortcodes =

* Display Speakers list in any post/pages:
<pre>[wep_speaker layout=1]<br>[wep_speaker layout=2]<br>[wep_speaker layout=3]<br>[wep_speaker layout=4]</pre>

* Display Session in any post/pages:
<pre>[wep_schedule layout=1]<br>[wep_schedule layout=2]<br>[wep_schedule layout=3]<br>[wep_schedule layout=4]</pre>

* Display Sponsors list in any post/pages:
<pre>[wep_sponsor layout=1]<br>[wep_sponsor layout=2]</pre>

== Installation ==

= Uploading in WordPress Dashboard =

1. Go to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select wp-event-partners.zip from your computer or Upload `wp-event-partners.zip` to the `/wp-content/plugins/` directory directly.
4. Click 'Install Now'
5. Activate the plugin through the 'Plugins' menu in WordPress.
