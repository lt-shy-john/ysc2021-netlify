== Description ==

One Page Conference is free WordPress theme for events, conferences, conclaves, and meetup. This theme is based on the Event and Conference Management WordPress Plugin, WPEventPartners. If you are the conferences organizers and looking to build one page conference website then this is the best theme for you. 

== Installation ==

	1. In your admin panel, go to Appearance > Themes and click the Add New button.
	2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
	3. Click Activate to use your new theme right away.


= Does this theme support any plugins? =

	One Page Conference includes support for Infinite Scroll in Jetpack.
	One Page Conference includes support for WPEventPartners plugin.



== Stylesheet ==
	Our theme is minimalistic theme for events so there is no dropdown in the navigation menu.

== Installation ==
	
	1. In your admin panel, go to Appearance > Themes and click the Add New button.
	2. Click Upload and Choose File, then select the one-page-conference.zip file. Click Install Now.
	3. Click Activate to use your new theme right away.
	4. Go to Customiser -> Home Page Setting -> Select - Static Front Page
	5. Install wpeventsparts wordpress plugin, then follow instruction from plugin.
	6. Create categories, pages and post
